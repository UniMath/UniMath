There are a couple of limitations
  • The De Bruijn index of a variable changes if the context changes. For statements like `‚àÄ x ‚àÄ y, P x y` where `P` is a propositional formula depending on `x` and `y`, this is no problem, because the context is fixed in `P`. As such, for such formulas this method is less suitable.
