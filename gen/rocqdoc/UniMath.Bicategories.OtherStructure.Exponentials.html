Definition make_is_cartesian_closed_bicat
           (B : bicat_with_binprod)
           (HB : is_univalent_2_1 B)
           (exp : B -> B ‚Üí B)
           (app : ‚àè (x y : B), exp x y ‚äó x --> y)
           (app2 : ‚àè (x y‚ÇÅ y‚ÇÇ : B)
                     (f g : y‚ÇÅ --> exp x y‚ÇÇ)
                     (Œ± : f ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ ==> g ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ),
                   f ==> g)
           (app2_eq : ‚àè (x y‚ÇÅ y‚ÇÇ : B)
                        (f g : y‚ÇÅ --> exp x y‚ÇÇ)
                        (Œ± : f ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ ==> g ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ),
                      app2 x y‚ÇÅ y‚ÇÇ f g Œ± ‚äó‚ÇÇ id‚ÇÇ (id‚ÇÅ x) ‚ñπ app x y‚ÇÇ = Œ±)
           (H : ‚àè (x y‚ÇÅ y‚ÇÇ : B)
                  (f g : y‚ÇÅ --> exp x y‚ÇÇ)
                  (Œ± : f ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ ==> g ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ)
                  (Œ≤‚ÇÅ Œ≤‚ÇÇ : f ==> g)
                  (p‚ÇÅ : Œ≤‚ÇÅ ‚äó‚ÇÇ id‚ÇÇ (id‚ÇÅ x) ‚ñπ app x y‚ÇÇ = Œ±)
                  (p‚ÇÇ : Œ≤‚ÇÇ ‚äó‚ÇÇ id‚ÇÇ (id‚ÇÅ x) ‚ñπ app x y‚ÇÇ = Œ±),
                Œ≤‚ÇÅ = Œ≤‚ÇÇ)
           (lam : ‚àè (x y‚ÇÅ y‚ÇÇ : B)
                    (f : y‚ÇÅ ‚äó x --> y‚ÇÇ),
                  y‚ÇÅ --> exp x y‚ÇÇ)
           (app_lam : ‚àè (x y‚ÇÅ y‚ÇÇ : B)
                        (f : y‚ÇÅ ‚äó x --> y‚ÇÇ),
                      invertible_2cell (lam x y‚ÇÅ y‚ÇÇ f ‚äó‚ÇÅ id‚ÇÅ x ¬∑ app x y‚ÇÇ) f)
  : is_cartesian_closed_bicat B.
Proof.
  intro x.
  use make_right_universal_arrow'.
  - exact HB.
  - exact (exp x).
  - exact (app x).
  - intros y‚ÇÅ y‚ÇÇ f g Œ±.
    simple refine (_ ,, _).
    + exact (app2 x y‚ÇÅ y‚ÇÇ f g Œ±).
    + exact (app2_eq x y‚ÇÅ y‚ÇÇ f g Œ±).
  - exact (H x).
  - intros y‚ÇÅ y‚ÇÇ f.
    simple refine (_ ,, _).
    + exact (lam x y‚ÇÅ y‚ÇÇ f).
    + exact (app_lam x y‚ÇÅ y‚ÇÇ f).
Defined.
