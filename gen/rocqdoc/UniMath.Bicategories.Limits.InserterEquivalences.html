Suppose, we have a diagram as follows
f₁ ----> p₁ ---- i₁ ----> x₁ y₁
g₁ ----> | | | | | | l₃ ≃ l₁ ≃ l₂ | | | | | | V V V
f₂ ----> p₂ ---- i₂ ----> x₁ y₂
g₂ ---->
where the columns are adjoint equivalences and both rows are inserters cones. If the top row is an inserter, then so is the bottom row.
