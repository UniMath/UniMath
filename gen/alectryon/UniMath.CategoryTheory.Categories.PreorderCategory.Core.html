(**************************************************************************************************

  The Category Generated by a Preorder

  For a preorder (a transitive and reflexive relation) ≤ on X, we can define a category, consisting
  of the elements of X, where a morphism between x and y is a proof that x ≤ y.
  An isomorphism between x and y is a pair of proofs that x ≤ y and y ≤ x. Therefore, if X is a set,
  the category is univalent if and only if x ≤ y and y ≤ x implies that x = y.

  Contents
  1. The category over a preorder [po_category]
  2. Univalence [po_category_is_univalent_iff_is_antisymm] [antisymm_po_univalent_category]

 **************************************************************************************************)
